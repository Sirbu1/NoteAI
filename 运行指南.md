# Android 项目运行详细指南（初学者版）

## 📋 前置准备

### 1. 检查 Android Studio 是否已安装

- 如果未安装，请先下载并安装 Android Studio
- 下载地址：https://developer.android.com/studio
- 安装时选择 "Standard" 安装模式

### 2. 确保已安装必要的组件

打开 Android Studio 后，检查是否已安装：
- Android SDK
- Android SDK Platform
- Android Virtual Device (AVD)

2. **在欢迎界面选择 "Open"**

   - 点击左侧的 **"Open"** 按钮

## ⏳ 第二步：等待 Gradle 同步

### 什么是 Gradle 同步？

Gradle 同步是 Android Studio 下载和配置项目依赖的过程。

### 同步过程

1. **自动开始同步**
   - 打开项目后，Android Studio 会自动开始 Gradle 同步
   - 底部状态栏会显示 "Gradle sync in progress..."

2. **等待完成**
   - 首次同步可能需要 **5-15 分钟**（取决于网络速度）
   - 请耐心等待，不要关闭 Android Studio

3. **同步成功的标志**
   - 底部状态栏显示：**"Gradle sync finished"**
   - 或者显示：**"BUILD SUCCESSFUL"**

### 如果同步失败

- 检查网络连接
- 查看错误信息（通常在底部的 "Build" 窗口）
- 尝试：`File` → `Invalidate Caches / Restart...` → `Invalidate and Restart`

---

## 📱 第三步：准备运行设备

你有两个选择：**使用真实手机** 或 **使用模拟器**

### 选项 A：使用真实 Android 手机（推荐）

#### 步骤 1：启用开发者选项

1. 打开手机的 **设置**
2. 找到 **关于手机**（About phone）
3. 连续点击 **版本号**（Build number）**7 次**
4. 会提示 "您已成为开发者"

#### 步骤 2：启用 USB 调试

1. 返回设置主界面
2. 找到 **开发者选项**（Developer options）
3. 打开 **USB 调试**（USB debugging）
4. 可能还需要打开 **USB 安装**（USB installation）

#### 步骤 3：连接手机到电脑

1. 使用 USB 数据线连接手机和电脑
2. 手机上会弹出提示，选择 **"允许 USB 调试"**
3. 勾选 **"始终允许来自这台计算机"**
4. 点击 **"确定"**

#### 步骤 4：验证连接

1. 在 Android Studio 中，查看顶部工具栏
2. 应该能看到你的设备名称出现在设备选择下拉框中

### 选项 B：使用 Android 模拟器（如果没有手机）

#### 步骤 1：创建虚拟设备

1. 在 Android Studio 中，点击工具栏的 **设备管理器图标**（手机图标）
   - 或者：`Tools` → `Device Manager`

2. 点击 **"Create Device"** 按钮

3. **选择设备型号**
   - 推荐选择：**Pixel 5** 或 **Pixel 6**
   - 点击 **"Next"**

4. **选择系统镜像**
   - 推荐选择：**API 34** (Android 14) 或 **API 33** (Android 13)
   - 如果显示 "Download"，点击下载（需要一些时间）
   - 点击 **"Next"**

5. **验证配置**
   - 检查配置信息
   - 点击 **"Finish"**

#### 步骤 2：启动模拟器

1. 在设备管理器中，找到刚创建的虚拟设备
2. 点击右侧的 **播放按钮**（▶️）
3. 等待模拟器启动（可能需要 1-2 分钟）
4. 模拟器启动后，会显示 Android 界面

---

## ▶️ 第四步：运行项目

### 步骤 1：选择运行设备

1. 在 Android Studio 顶部工具栏，找到 **设备选择下拉框**
   - 应该显示 "No devices" 或你的设备名称

2. **点击下拉框**，选择：
   - 你的真实手机（如果已连接）
   - 或你的模拟器（如果已启动）

### 步骤 2：运行应用

**方法一：使用运行按钮（最简单）**

1. 点击顶部工具栏的 **绿色运行按钮**（▶️）
   - 或者按快捷键：`Shift + F10`（Windows）

2. 等待编译和安装
   - 底部会显示 "Building..." 和 "Installing..."
   - 首次运行可能需要 1-3 分钟

3. **应用会自动安装并启动**
   - 在手机上或模拟器上会看到应用打开

**方法二：使用菜单**

1. 点击菜单：`Run` → `Run 'app'`
2. 或者按快捷键：`Shift + F10`

### 步骤 3：查看运行结果

1. **在设备上**
   - 应用会自动打开
   - 你应该看到 "智能笔记" 应用的主界面

2. **在 Android Studio 中**
   - 底部会打开 "Run" 窗口
   - 显示应用的运行日志
   - 如果看到 "BUILD SUCCESSFUL"，说明运行成功

## ❗ 常见问题解决

### 问题 1：找不到设备

**症状**：设备选择下拉框显示 "No devices"

**解决方法**：
- 检查手机 USB 连接
- 确认已启用 USB 调试
- 尝试重新插拔 USB 线
- 在手机上重新授权 USB 调试

### 问题 2：Gradle 同步失败

**症状**：底部显示红色错误信息

**解决方法**：
1. 检查网络连接
2. 尝试：`File` → `Invalidate Caches / Restart...`
3. 检查 `gradle.properties` 文件中的代理设置

### 问题 3：编译错误

**症状**：运行时报错，显示红色错误信息

**解决方法**：
1. 查看 "Build" 窗口的错误详情
2. 检查是否缺少 SDK
3. 尝试：`Build` → `Clean Project`，然后 `Build` → `Rebuild Project`

### 问题 4：应用安装失败

**症状**：显示 "Installation failed"

**解决方法**：
1. 检查设备存储空间
2. 卸载设备上已有的同名应用
3. 检查设备是否允许安装未知来源的应用

### 问题 5：模拟器启动慢

**症状**：模拟器启动需要很长时间

**解决方法**：
- 这是正常的，首次启动需要 1-2 分钟
- 确保电脑有足够的内存（建议 8GB 以上）
- 可以在 AVD 设置中减少 RAM 分配

## 💡 小贴士

1. **首次运行较慢**：第一次运行项目时，Android Studio 需要下载依赖和编译，会比较慢，这是正常的。

2. **保持网络连接**：Gradle 同步和下载依赖需要网络，确保网络畅通。

3. **使用真实设备更快**：如果有 Android 手机，使用真实设备比模拟器运行更快。

4. **查看日志**：如果应用有问题，查看底部的 "Logcat" 窗口，可以看到详细的错误信息。

5. **保存工作**：运行前记得保存所有文件（`Ctrl + S`）。
